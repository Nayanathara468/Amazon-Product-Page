Amazon Product Page UI Clone

Overview

This project is a simple Amazon Product Page UI Clone developed using HTML, CSS, and JavaScript. It recreates the layout and appearance of an Amazon product page to practice front-end web development skills.

Features

- Amazon-style header and navigation bar
- Search bar
- Product image section
- Product title and ratings
- Price and product description
- Add to Cart button
- Buy Now button
- Responsive design for desktop and mobile
- Simple JavaScript interactions

Technologies Used

- HTML5
- CSS3
- JavaScript

Project Structure

Amazon-Clone/
│── index.html
│── style.css
│── script.js
│── amazon-logo.png
│── headphones.jpg
│── README.md

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click index.html or open it in any web browser.

Learning Objectives

- Build a responsive web page.
- Practice HTML, CSS, and JavaScript.
- Understand UI cloning techniques.
- Improve front-end development skills.
